  1. Que es un sistema de control de versiones?

    Es una herramienta que nos permite gestionar los cambios en nuestros archivos de codigo, para controlar y mejorar el flujo de trabajo entre los equipos de desarrollo

  2. Cual es la diferencia entre git y github?

    Git es la herramienta de control de versiones, Github es un servicio que permite almacenar y gestionar repositorios git

  3. Cómo creo una rama en git?

    Una rama en git se crea con el siguiente comando:

    git branch [nombre_rama]

  4. Cómo clono un repositorio?

    Un repositorio en git se clona con el siguiente comando:

    git clone [direccion_repositorio_a_clonar]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    Tomar los ultimos cambios del repositorio se hace con este comando:

    git pull

  6. Cómo puedo gener un tag?

    Se puede poner un tag con este comando:

    git tag [nombre_tag]

  7. Cómo hago una mezcla de ramas?

    Combinar ramas se hace con este comando:

    git merge [nombre_rama_a_combinar]

  8. Cómo puedo moverme a un commit especifico?

    Se puede seleccionar un commit en particular con este comando:

    git checkout [id_del_commit]
